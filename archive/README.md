# Archive

This directory keeps non-final materials out of the active v0.3 repository root.

- `legacy_inputs/`: early unversioned input files kept for comparison.
- `v0.1_inputs/`: v0.1 task, reference, task-pool, rubric, and run-plan files.
- `v0.1_results/`: v0.1 raw runs, normalized outputs, reports, score CSVs, and pilot review workbook.
- `v0.2_inputs/`: v0.2 private-reference and rubric drafts.
- `v0.2_outputs/`: v0.2 readiness audit and manual scoring workbook.
- `v0.3_superseded_results/`: v0.3 retry outputs not used by the final scoring tables.
- `unused_code/`: scripts not used by the v0.3 pipeline.

The active root should only expose v0.3 paper-facing inputs, runs, scores, outputs, and scripts.
