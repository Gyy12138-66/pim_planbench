# PIM-PlanBench v0.4 修订路线（回应 ARR 评审）

- 评审：Ed11（Overall 2，Soundness 2）、PdEo（2.5/2.5）、d2gd（3/3）
- 主线不变：**快速响应、省算力的 planning-stage 基准**。执行永不进入评测流程。
- 核心策略：把"廉价代理的效度证明"升级为论文的科学主张（Ed11 要的可证伪假设），
  用测量学效度框架组织：区分度 → 抗套模板 → 收敛效度 → 预测效度。

## 坚持的立场（rebuttal 口径）

1. execution-free 是身份不是缺陷：执行型基准已有（PDEBench、PINNacle、ScienceAgentBench），
   空缺的恰是训练前的廉价早期信号；效度子集（#5）补上证据缺口后，该点从弱点翻为卖点。
2. "小而锋利"是合法品类：饱和才是死罪（#1 解决），规模适度增至 ~40 并给选题方法论即可，
   不追求大而全（大 = 参考撰写成本 + 陷阱稀释 + 评测变贵，全部背离主线）。
3. 私有参考不全公开：污染控制的社区惯例（GSM1k 式保留集）；答案是治理政策，不是公开。
4. 自动评分优先不变：LLM-judge 可作第二意见评分器并报告一致性，是补充不是替换。
