## 设计原则（同 item_analysis §3）

- **P1 决策支点**：每题一个二元、后果严重、模板绕不过的建模决策，cap 钉住它
- **P2 可核验断言**：支点尽量以定量断言暴露，供断言核验评分器机器检查
- **P3 反模板机制**：欠定规格逼假设声明 / 干扰物理项逼筛选 / 算力预算逼取舍
- **P4 私有参考同步**：failure_traps / facet_cap_rules / critical_points 同步更新
